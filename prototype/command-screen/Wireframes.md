# Wireframes — 指挥大屏布局变体（低保真 / ASCII）

> 只关注结构与信息层级，不涉及视觉。每个变体标注：布局策略、信息优先级（第一眼/第二眼/第三眼）、区块面积占比。桌面 + 移动兜底。
> 数据来源接口见 DesignBrief §5。

---

## 变体 A —「中央战场」三栏驾驶舱
**布局策略**：左数据 / 中地图 / 右动态流；顶栏 + 底字幕。最像"作战指挥墙"，中心留给空间态势。

```
┌──────────────────────────────────────────────────────────────────────┐
│ 顶栏: 舆情指挥中枢 | 实时时钟(Mono) | 采集心跳● | 连接态:轮询中        │ 高 6%
├───────────────┬───────────────────────────────┬──────────────────────┤
│ 左栏 24%       │ 中栏 52%                        │ 右栏 24%             │
│ ┌───────────┐ │ ┌───────────────────────────┐ │ ┌──────────────────┐ │
│ │ KPI:今日新增│ │ │ 全局风险水位环 + 概述       │ │ │ 实时快讯流 LIVE   │ │
│ │ 读数 1,284 │ │ │ (RiskGauge + 关键数)       │ │ │ [时间][风险][摘要]│ │
│ ├───────────┤ │ └───────────────────────────┘ │ │ ...滚动...        │ │
│ │ KPI:高风险  │ │ ┌───────────────────────────┐ │ │ ...              │ │
│ │ 读数 37    │ │ │ 地区分布条 RegionBars TOP10│ │ └──────────────────┘ │
│ ├───────────┤ │ │ (横向条形 + 省名)           │ │ ┌──────────────────┐ │
│ │ KPI:未处置  │ │ └───────────────────────────┘ │ │ 预警滚动 AlertScroll│ │
│ │ 预警 12    │ │ ┌───────────────────────────┐ │ │ critical=红 ...   │ │
│ ├───────────┤ │ │ 传播态势 PropagationTree    │ │ │ high=橙 ...       │ │
│ │ 情感分布    │ │ │ (ECharts graph 重点事件)   │ │ └──────────────────┘ │
│ │ Sentiment   │ │ └───────────────────────────┘ │                      │
│ │ Donut      │ │                                │ │                      │
│ └───────────┘ │                                │ │                      │
├───────────────┴───────────────────────────────┴──────────────────────┤
│ 底字幕 Ticker: 关键事件标题横向滚动 ……                                    │ 高 4%
└──────────────────────────────────────────────────────────────────────┘
```
- **第一眼**：中栏（风险水位 + 地区态势 + 传播图）——空间感最强。
- **第二眼**：左栏今日新增 / 高风险 / 未处置预警（指挥者最关心数字）。
- **第三眼**：右栏实时流与预警（持续跳动吸引余光）。
- **适用**：强调"地理/传播空间态势"的指挥场景。

## 变体 B —「双翼对称」顶栏 KPI + 三栏网格
**布局策略**：顶部一条全宽 KPI 带，下方三栏（左环图 / 中趋势英雄 / 右告警），底部词云带。最均衡、信息密度高。

```
┌──────────────────────────────────────────────────────────────────────┐
│ 顶栏: 舆情指挥中枢 | 时钟 | 心跳● | 连接态                                        │
├──────────────────────────────────────────────────────────────────────┤
│ KPI 带(全宽): [总舆情 48,201] [今日 +1,284] [高风险 37] [未处置 12] [事件 9] │ 高 12%
├──────────────┬─────────────────────────┬──────────────────────────────┤
│ 左 28%        │ 中 44%                   │ 右 28%                       │
│ ┌──────────┐ │ ┌─────────────────────┐ │ ┌──────────────────────────┐│
│ │ 风险水位环 │ │ │ 趋势折线 TrendLine   │ │ │ 预警滚动 AlertScroll     ││
│ │ RiskGauge│ │ │ │ (近7日舆情量)        │ │ │ critical/high/med/low    ││
│ ├──────────┤ │ └─────────────────────┘ │ └──────────────────────────┘│
│ │ 情感分布   │ │ ┌─────────────────────┐ │ ┌──────────────────────────┐│
│ │ Donut    │ │ │ 地区分布条 RegionBars │ │ │ 实时快讯流 LiveFeed      ││
│ └──────────┘ │ │ │ TOP10               │ │ │ LIVE 滚动                ││
│              │ └─────────────────────┘ │ └──────────────────────────┘│
├──────────────┴─────────────────────────┴──────────────────────────────┤
│ 词云带(全宽): 热点词云 KeywordCloud TOP10 ……                              │ 高 16%
├──────────────────────────────────────────────────────────────────────┤
│ 底字幕 Ticker                                                              │
└──────────────────────────────────────────────────────────────────────┘
```
- **第一眼**：顶部 KPI 带（五个核心数，一眼掌握全局）。
- **第二眼**：中栏趋势折线（时间维度态势）。
- **第三眼**：右栏告警/快讯（动态）、左栏分布图（结构）。
- **适用**：信息密度高、强调"核心数字 + 时间趋势"的汇报场景。

## 变体 C —「非对称聚焦」左英雄 + 右动态流
**布局策略**：左 60% 放英雄区（地区态势 + 趋势），右 40% 竖向堆叠动态流。最聚焦、最"沉浸"。

```
┌──────────────────────────────────────────────────────────────────────┐
│ 顶栏: 舆情指挥中枢 | 时钟 | 心跳● | 连接态                                        │
├───────────────────────────────────┬──────────────────────────────────┤
│ 左 60%                             │ 右 40%                          │
│ ┌───────────────────────────────┐ │ ┌──────────────────────────────┐│
│ │ 风险水位环 + 概述 (左上角浮起)   │ │ │ 实时快讯流 LIVE              ││
│ ├───────────────────────────────┤ │ │ [时间][风险][摘要]           ││
│ │ 地区分布条 RegionBars TOP10    │ │ │ 滚动...                      ││
│ │ (大尺寸横向条形)                │ │ ├──────────────────────────────┤│
│ ├───────────────────────────────┤ │ │ 预警滚动 AlertScroll         ││
│ │ 趋势折线 TrendLine (近7日)      │ │ │ critical=红 high=橙          ││
│ │ (英雄宽度)                      │ │ ├──────────────────────────────┤│
│ ├───────────────────────────────┤ │ │ 传播态势 PropagationTree     ││
│ │ 传播态势 PropagationTree        │ │ │ (重点事件节点)               ││
│ │ (占左栏下半)                    │ │ ├──────────────────────────────┤│
│ │                                │ │ │ 情感分布 Donut(小)           ││
│ └───────────────────────────────┘ │ └──────────────────────────────┘│
├───────────────────────────────────┴──────────────────────────────────┤
│ 底字幕 Ticker                                                              │
└──────────────────────────────────────────────────────────────────────┘
```
- **第一眼**：左栏地区态势（大条形）+ 风险环。
- **第二眼**：左栏趋势折线（时间态势）。
- **第三眼**：右栏整列动态流（快讯→预警→传播→情感，持续刷新）。
- **适用**：单主屏、强调"空间态势 + 实时流"的沉浸指挥。

## 移动/小屏兜底（三变体通用）
<1280px：右栏/词云带堆叠到主体下方，KPI 带换行；
<768px：单列竖向滚动，图表降为纵向卡片，关闭扫描线动效。
