# 指挥大屏 · 地图自动轮播下钻（方案 B）实现报告

> 日期：2026-07-22　|　改动文件：`frontend/src/components/command-screen/ChinaMap.vue`（纯前端，无后端改动）
> 需求：全域舆情热力 · 地区 TOP 地图支持「自动依次放大到各省 → 逐市区高亮显示 → 回全国 → 下一省循环」的动画轮播；轮播控制面板默认隐形、可展开/收缩。

## 一、实现方案

### 1. 自动轮播状态机（tour）
复用既有下钻能力（`drillTo` / `backToCountry` / GeoJSON 缓存 / `registerMapOnce`），新增串行控制器 `startTour()`：

```
for each prov in 轮播列表(循环):
  drillTo(prov)              # 放大下钻到该省（加载市级 GeoJSON + 拉取市/县分布）
  sleep(省停留 1.6s)
  for each city in 该省市区:
    dispatchHighlight(i)      # 逐个高亮该市区 + 弹出 tooltip（showTip）
    sleep(市停留 0.9s)
  dispatchDownplayAll()       # 取消高亮
  backToCountry()             # 回到全国
  sleep(全国停留 1.3s)
```

关键工程细节：
- **可中断令牌 `tourToken`**：每次 `startTour` 自增，循环中每步校验；`stopTour` / 组件卸载自增令牌并清理在途定时器，杜绝泄漏与「卸载后还在跑」。
- **暂停可恢复**：`togglePause()` 通过 `waitWhilePaused()` 挂起循环，恢复后从断点继续。
- **手动点击即退出**：`onMapClick` 中若 `tourActive` 则 `stopTour()`，退回手动交互模式。
- **预取 GeoJSON**：`startTour` 前 `prefetchTourGeo()` 并行拉取轮播省份边界并塞入 `geoCache`，避免轮播到时才联网卡顿（河北本地打包，其余按需 DataV 并缓存）。
- **下钻「推近」动画**：`.cs-map-wrap.is-province .cs-map` 加 `cs-tour-zoom`（scale 1.08→1 + 透明度淡入，0.55s），视觉模拟「放大聚焦」。
- **reduced-motion 友好**：`@media (prefers-reduced-motion: reduce)` 关闭放大与面板入场动画。

### 2. 隐形可折叠轮播控制面板（用户核心诉求）
右下角默认**仅一个小圆点**（`cs-tour-fab`，`opacity:0.5`，30×30 圆形「▶」），几乎不干扰大屏视觉；点击展开为玻璃拟态卡片（`cs-tour-panel`，226px，blur 背景）：

| 区域 | 内容 |
|------|------|
| 头部 | 「自动轮播下钻」+ ×（收起回隐形圆点） |
| 控制 | ▶ 开始 / ⏸ 暂停·继续 / ■ 停止 |
| 轮播列表 | 当前省份 chips（当前轮播到的省高亮 `active`）；每个 chip 可 × 移出；底部 `+ 添加省份` 下拉从全部 34 省候选中增选 |
| 状态 | 「轮播中 · 河北省」/「已暂停」 |

- **持久化**：轮播列表保存在 `localStorage['cmd-tour-provinces']`，刷新后保留自定义；首次使用（无自定义）自动以「有数据的省」（当前=河北省）初始化。
- **候选列表动态**：`candidateProvinces` = 全部省名 − 已选，已选满则下拉隐藏。

## 二、数据口径（真实，不伪造）
- 市级分布取自既有 `GET /api/dashboard/region-children`（河北 290 条 / 12 市区，含大厂县等已由代码前缀推导修正）。
- 轮播到无数据省（如除河北外的其他 33 省）时：放大显示省级轮廓 + 徽标「该省暂无细分数据」，城市级逐个点亮环节为空，不报错（接口 404 优雅降级）。
- 当前库数据全在河北省 → 首版轮播即饱满（逐个点亮 12 市区 + tooltip 真实数字）；等数据铺开其他省，把省名加入轮播列表即可，无需改结构。

## 四、已知限制与后续
- 仅河北省有真实市级数据；其他省轮播为空轮廓（与「数据源将不限于河北」的演进方向一致，待数据补齐即饱满）。
- 雄安新区的市级数据并入保定市（标准 GeoJSON 无独立要素），地图无独立着色，总数/tooltip 不受影响。
- 轮播「放大」为离散切换 + 缩放淡入（非连续推镜），是大屏类项目通用做法，观感足够「指挥感」。
- 如需全国 34 省全轮播，把候选省逐个加入列表即可；也可后续加「跳过空省」开关。
